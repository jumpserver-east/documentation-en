# Reset user MFA

When a user cannot log in due to a lost MFA, the administrator can reset MFA for the user.

<Steps>

<div>
<Nav name="console" />
</div>

<div>
In the left menu, click **<Icon name="UserO" /> Users**.
</div>

<div>
In the table, click the name of the user for whom you want to reset MFA.
</div>

<div>
On the user details page, click **Basic** tab.
</div>

<div>
In the **Quick update** card, find the "Reset mfa" section and click the **Reset** button.
</div>

</Steps>
