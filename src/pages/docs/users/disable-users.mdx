# Disable users

## Disable a user

<Steps>

<div>
<Nav name="console" />
</div>

<div>
In the left menu, click **<Icon name="UserO" /> Users**.
</div>

<div>
In the table, click the name of the user you want to disable.
</div>

<div>
On the user details page, click **Basic** tab.
</div>

<div>
In the **Quick update** card, turn off the **Active** toggle to disable the user.
</div>

</Steps>

## Bulk disable users

<Steps>

<div>
<Nav name="console" />
</div>

<div>
In the left menu, click **<Icon name="UserO" /> Users**.
</div>

<div>
In the table, check the users you want to disable.
</div>

<div>
Above the table, select **Actions <Icon name="chevron-down" />** button, then click **<Icon name="Ban" /> Disable selected**.
</div>

</Steps>
