# Export users to file

This topic describes how to export users.

## Export all users
<Steps>

<div>
<Nav name="console" />
</div>

<div>
Navigate to **Users**, in the upper right corner of the user list, click the <Icon name="download"/> .
</div>

<div>
In the popup, select the file type and export range.
<img src="/images/docs/admin-guides/users/export-all-users.png" alt="image" width="450"/>
</div>

<dev>
Click **Confirm**.
</dev>

</Steps>

## Export selected users
<Steps>

<div>
<Nav name="console" />
</div>

<div>
Navigate to **Users**.
</div>

<div>
In the user list, select users for export, in the upper right corner of the user list, click the <Icon name="download"/>.
</div>

<div>
In the popup, select the file type and export range.
<img src="/images/docs/admin-guides/users/export-selected-users.png" alt="image" width="450"/>
</div>

<dev>
Click **Confirm**.
</dev>
</Steps>

## Export filtered users
<Steps>

<div>
<Nav name="console" />
</div>

<div>
Navigate to **Users**.
</div>


<div>
In the upper right corner of the user list, type search criteria in the search box, click the <Icon name="download"/>.
</div>

<div>
In the popup, select the file type and export range.
<img src="/images/docs/admin-guides/users/export-filtered-users.png" alt="image" width="450"/>
</div>

<dev>
Click **Confirm**.
</dev>
</Steps>
